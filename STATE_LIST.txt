## Estados Customizados (0x80+)

| State | Nome | Descricao |
|-------|------|-----------|
| 0x80 | STATE_LOGO_SKIP | Pular animacao do logo (transicao direta para menu) |
| 0x81 | STATE_SONG_SELECT | Tela de selecao de musicas |
| 0x82 | STATE_SONG_SELECT_B | Transicao (fade in) para song select |
| 0xFF | STATE_EXIT | Fechar o jogo (fadeout preto + quit) |

## Fluxo Normal do Jogo

WARNING (0x01) -> WARNING_ANIM (0x02) -> WARNING_END (0x03)
  -> LOGO_ENTER (0x04) -> LOGO_UPDATE (0x05)
  -> RESET_FLOW (0x06) -> MENU_IDLE (0x08) -> MENU_INPUT_WAIT (0x09)
  -> MENU_ENTER (0x0A) -> MENU_INPUT (0x0B) [loop]

MENU_INPUT + UL 2x -> MENU_FADE_IN (0x07) -> SONG_SELECT (0x81)
MENU_INPUT + DR 2x -> EXIT (0xFF) [fadeout preto + quit]
SONG_SELECT -> GAME_INIT (0x0E) -> GAMEPLAY (0x0F)
