Notas da Cheat Engine

  • States 0x03 e 0x06 existem no dispatcher mas nao produzem nenhuma cena visivel nem trigger — provavelmente vestigios do arcade.
  • States 0x07-0x09 (HowToPlay) sao funcionais no executavel mas nao estao no fluxo normal do menu PC. Parecem ter sido desativados intencionalmente.
  • State 0x0C faz reboot — volta pro boot. Confirmado via CE.
  • State 0x13 (AutoExit) redireciona para 0x04. Funcao exata ainda desconhecida.
  • State 0x16 (StageBreak) tem condicao dupla: (LifeGauge == 0 && StageBreakOption == true) OU (Miss Combo == 51 && StageBreakOption == false).
  • State 0x22 (GameOptionExit) possivelmente salva opcoes. Nao confirmado.
